  # Ассистрал
[English](README.md) | **Русский**

Простая WebView-обёртка для Mistral Le Chat, блокирующая ненужные ссылки для сосредоточенного общения.

## Возможности

- 🔒 **Защита конфиденциальности**: Блокирует ненужные ссылки и отслеживание
- 🎯 **Минималистичный интерфейс**: Чистый чат без отвлекающих элементов браузера
- 🔄 **Переключение блокировки**: Легкий переход между режимами (с блокировкой/без блокировки)
- 📁 **Поддержка файлов**: Загрузка файлов для анализа и диалога
- 🎤 **Голосовой ввод**: Поддержка голосовых сообщений
- 📱 **Легковесность**: Минималистичное приложение с базовым функционалом

## Использование

Приложение не поддерживает вход через аккаунты Apple, Microsoft или Google.
Для первоначальной регистрации используйте веб-браузер.

Вы можете включать или отключать блокировку, нажав на кнопку в правом верхнем углу.
Чтобы скрыть кнопку, проведите вверх для более чистого вида экрана.

## Лицензия

Приложение распространяется под лицензией **GPLv3**.

В приложении используются компоненты:
- Части **GMaps WV** ([ссылка](https://gitlab.com/divested-mobile/maps)), распространяемые под **GPLv3**
