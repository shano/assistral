<p><strong>Ассистрал</strong> — простая WebView-обёртка для Mistral Le Chat. Блокирует все ненужные для работы ссылки.</p>
<p><strong>Важно:</strong> Вход через аккаунты Apple, Microsoft или Google не поддерживается. Для первоначальной регистрации используйте веб-браузер.</p>
<p><strong>Управление:</strong></p>
<ul>
<li><p>Включение/отключение блокировки — кнопка в правом верхнем углу.</p></li>
<li><p>Чтобы скрыть кнопку, проведите вверх.</p></li>
</ul>
